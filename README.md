# 💰 Yari - The Digital Economy for Campus Life

A vibrant, animated landing page for the Yari campus economy platform. Built with a playful, Duolingo-inspired design system.

## ✨ Features

- 🎨 **Vibrant Design:** Playful colors and animations inspired by modern learning apps
- 📱 **Mobile-First:** Fully responsive, looks great on all devices
- ⚡ **Smooth Animations:** Scroll-triggered reveals, floating elements, and micro-interactions
- 🎯 **Conversion-Optimized:** Clear CTAs and engaging copy
- 🚀 **Fast & Lightweight:** Pure HTML, CSS, and JavaScript (no frameworks)

## 🎯 Quick Start

1. **Update Google Form Link:**
   - Open `index.html`
   - Find `YOUR_GOOGLE_FORM_LINK` (appears 4 times)
   - Replace with your actual Google Form URL

2. **Test Locally:**
   - Double-click `index.html`
   - Check that everything looks good

3. **Deploy:**
   - See `DEPLOYMENT_GUIDE.md` for detailed instructions
   - Recommended: GitHub Pages or Netlify

## 🔧 Customization

### Update Student Count
```html
<!-- In index.html, line 34 -->
<span>100+ students already joined waitlist</span>
```

### Update Balance Amount
```javascript
// In script.js, line 146
const target = 120; // Change to your desired amount
```

### Update Stats
```html
<!-- In index.html, stats section -->
<div class="stat-value" data-target="100">0</div>
```

### Add Your Logo
Replace the emoji logo (💰) with an image:
```html
<!-- Replace in index.html -->
<div class="logo-icon">
    <img src="your-logo.png" alt="Yari" style="width: 40px; height: 40px;">
</div>
```
