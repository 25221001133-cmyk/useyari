# Cool Ride Website

Customer-facing makeup landing page for **Cool Ride**, a startup ride service.

## Files

- `index.html` - Single-page responsive website (HTML + CSS + minimal JS)
- `CNAME` - Custom domain configuration for GitHub Pages

## Run locally

Open `index.html` in your browser.

## Customize quickly

- Update hero copy and CTA text in `index.html`
- Replace `mailto:hello@coolride.com` with your real support/waitlist address
- Edit section content (features, reviews, FAQ) to match your launch city and offer

## Deployment

This repository is static and ready for GitHub Pages deployment.
